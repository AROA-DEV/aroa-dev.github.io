Using the Terminal

How to use Terminal

Linux will mainly be interacting within the terminal, the Desktop environment, or GUI, using the desktop environment most times will not allow performing actions that need root(admin) level privileges. Which means that all Linux users will have to learn to use some of the basic commands such as [ ls ] to list directories and files, cd to move into one of the listed directories [ Sudo -other command - ] to perform it as root, [ rm ] remove a file, [ man ] shows the manual for a command
