Dual Boot

What is Dual Booting?

Dual Booting is a way of shearing a machine between two different operating systems Ex. Linux and Windows, this process will divide the disk between both operating systems, is recommended to use a disk of 250GB as a minimum since windows have a minimum space of around 160GB.

for kali large as an example you will need at least 40GB for all the tools and 10GB extra for other data you m, may need.

1
Get the things tou need

First, you will need to choose your operating system and download the ISO, Get Balenaetcher (all operating systems), or Rufus (only Windows) to flash the iso into a USB, preferably 16GB or larger. Once you start the process no matter which app you use all the data on the USB will be lost.

2
Flash the ISO

1. Select your ISO, select the USB ( !!! Be careful with the USB you choose you may lose data !!! ), 2. Select your ISO and pres Flash

3
Divide the disk

only if you have one disk if you have two, you can use the second disk to only run Linux

Now open disk manager (Windows)you can find it by pressing right-clicking the windows logo on your taskbar and pressing "disk manager" then right-click on disk one (I'm shrinking disk two since I have free space there) and click shrink, a popup will appear with the options of size and chouse the amount of space you need for your Linux OS

4
Setup the Boot menu

To open the boot menu you will have to research your pc motherboard and the key to open the boot menu, once you know the key, you will restart the pc while at the same time spamming the key.

Then a boot menu will pop up each manufacterer has a diferent one but the most comon will look like the one in the photo above, muve to the boot section with the arows. And put the USB as the first boot option and save and exit the boot menu

Install Menu
