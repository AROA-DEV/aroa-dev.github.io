BARE METAL INSTALL

1
Get the things tou need

First, you will need to choose your operating system and download the ISO make, get vm workstation, virtual box and install theme

2
Start your first VM

Get your hosting software and install it, then each software has a different process to start a vm, so you will have to check their documentation below you also have some video tutorials

3
Setup the Boot menu

To open the boot menu you will have to research your pc motherboard and the key to open the boot menu, once you know the key, you will restart the pc while at the same time spamming the key.

Then a boot menu will pop up each manufacturer has a different one but the most common will look like the one in the photo above, move to the boot section with the arrows. And put the USB as the first boot option and save and exit the boot menu

4
Install Menu

Now if you have followed all the steps the install screen for the ISO should have popped up, just follow the instructions [ if you have any difficulty looking up the documentation for the ISO. We are not able to make a tutorial for each of all the variants because of the amount of themes]

5
Desktop Environment

During some of the installs, you will be able to choose a desktop environment, these will cause affect performance, xface is the lighter one and also the simplest. Gnome is the best looking one in my opinion but it will slightly reduce performance but it might be worth the cost if you like to customize your visual experience
