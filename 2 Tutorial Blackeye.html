Blackeye

Blackeye is a package of login web pages, that can be deployed with Ngrok

Download the package with:

    

    (kali⭐kali)-[~]
    └─# cd Desktop
    
    (kali⭐kali)-[~]
    └─#  git clone https://github.com/AROA-DEV/Blackeye.git
2

Start Ngrok

    

    (kali⭐kali)-[~]
    └─# cd Desktp
    
    (kali⭐kali)-[~]
    └─# wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
    
    (kali⭐kali)-[~]
    └─# unzip /path/to/ngrok.zip
    
    (kali⭐kali)-[~]
    └─# ngrok config add-authtoken [ your personal authotoken ]
    
    (kali⭐kali)-[~]
    └─# ngrok http 8080

Push the Blackeye page

    

┌──(anonimus㉿kali)-[~]
└─$ cd Desktop                                                               
                                                                                                                   
┌──(anonimus㉿kali)-[~/Desktop]
└─$ git clone https://github.com/AROA-DEV/Blackeye.git             

┌──(anonimus㉿kali)-[~/Desktop]
└─$ cd Blackeye 
                                                                                                                   
┌──(anonimus㉿kali)-[~/Desktop/Blackeye]
