Wifite

Wifite is a wifi cracking Tools for Linux, it comes as standard with kali linux but you will net to install its dependencies.

Keep in mind that you will only be able tu use these tool with a wifi card that allows monitor mode, if your integrated wifi card does not alow monitor mode you can purchase a usb wifi card that supports it, we recommend the AWYS036ACM from ALFA NETWORK

2


Installing wifite

Run wifite


    (kali⭐kali)-[~]
    └─# sudo -i 

    (root⭐root)-[~]
    └─# git clone https://github.com/derv82/wifite2.git

    (root⭐root)-[~]
    └─# cd wifite2
                           
    (root⭐root)-[~]
    └─# sudo ./Wifite.py

Installing dependencies


    # If you are in the same terminal just follow these steps if not go to the wifite directory 

    (root⭐root)-[~]
    └─# pip3 install -r requirements.txt

    (root⭐root)-[~]
    └─# apt update 

    (root⭐root)-[~]
    └─# apt upgrade

    # these should install all of the dependencies automatically if it doesn't follow these
                           
    (root⭐root)-[~]
    └─# sudo -i 
    
    hcxdumptool:
    
    (root⭐root)-[~]
    └─# sudo apt install hcxdumptool 
    
    # hcxpcaptool:
    
    (root⭐root)-[~]
    └─# apt install hcxtools
    
    # pyrit:
    
    (root⭐root)-[~]
    └─# sudo apt-get install libpcap-dev
    
    (root⭐root)-[~]
    └─# sudo apt-get install python2.7-dev libssl-dev zlib1g-dev libpcap-dev
    
    (root⭐root)-[~]
    └─# git clone https://github.com/JPaulMora/Pyrit.git
    
    (root⭐root)-[~]
    └─# cd Pyrit
    
    (root⭐root)-[~]
    └─# sudo python setup.py clean
    
    
    (root⭐root)-[~]
    └─# sudo python setup.py build
    
    (root⭐root)-[~]
    └─# sudo python setup.py install
